    ## Calcul de la Défense (DEF) Totale

    ### **1. Défense de Base (sans équipement)**

    - **VIT de Base** : 161 (156 + 5)
    - **Niveau** : 99

    **Formule pour les niveaux 90-99 :**
    \[
    \text{DEF de Base} = \text{floor}(161 \times 1.5) + 99 + 18 + \text{floor}\left(\frac{99 - 89}{2}\right) = 241 + 99 + 18 + 5 = 363
    \]

    ### **2. Bonus DEF des Traits de Job**

    - **Trait VI (PLD91)** : +72 DEF

    ### **3. Bonus DEF des Gifts de Job**

    - **Physical Defense Bonuses** :
      - +15 DEF
      - +23 DEF
      - +30 DEF
      - +38 DEF
    - **Total** : 15 + 23 + 30 + 38 = **106 DEF**

    ### **4. Défense Brute des Équipements**

    - **Total DEF Brute** : 1,033 DEF

    ### **5. DEF via VIT**

    - **VIT Totale** : 161 (base) + 324 (équipement) = **485 VIT**
    - **Calcul de DEF via VIT** :
      \[
      \text{DEF via VIT} = \text{floor}(485 \times 1.5) = 727 \text{ DEF}
      \]
    - **DEF via VIT de Base** (sans équipement) :
      \[
      \text{DEF via VIT de Base} = \text{floor}(161 \times 1.5) = 241 \text{ DEF}
      \]
    - **Augmentation de DEF via VIT due à l'équipement** :
      \[
      727 - 241 = 486 \text{ DEF}
      \]

    ### **6. Calcul Final de la DEF Totale**

    \[
    \text{DEF Totale} = \text{DEF de Base} + \text{Bonus DEF Traits} + \text{Bonus DEF Gifts} + \text{DEF Brute Équipements} + \text{Augmentation DEF via VIT}
    \]
    \[
    \text{DEF Totale} = 363 + 72 + 106 + 1,033 + 486 = 2,060 \text{ DEF}
    \]
