# Analyse Comparative des Sets Défensifs : Sakpata vs Chevalier pour la Résistance Magique

## Résultats des Tests de Dégâts

Tests effectués sur Spurned Elementalist (niveau 122-123) :

- Set Sakpata : 49.84 dégâts moyens reçus
- Set Chevalier : 98.04 dégâts moyens reçus

## Comparaison des Statistiques Clés

Set Sakpata :

- Magic Evasion : 727
- Magic Defense Bonus : 36
- Elemental Resist : 15
- INT : 235

Set Chevalier :

- Magic Evasion : 667
- Magic Defense Bonus : 47
- Elemental Resist : 15
- INT : 256

## Analyse Détaillée des Mécaniques

### 1. Magic Evasion (Différence +60 pour Sakpata)

- Plus importante statistique pour la différence de performance
- +2 Magic Evasion = +1% de résistance (sous 50% de taux de résist)
- +2 Magic Evasion = +2% de résistance (au-dessus de 50% de taux de résist)
- Combinée avec Elemental Resist et Carrier's Sash, permet d'obtenir des "full resists" (1/8 des dégâts)
- Sans Elemental Resist : maximum "partial resist" (1/4 des dégâts)

### 2. Magic Defense Bonus (Différence +11 pour Chevalier)

- Utilisé dans la formule : (100 + Magic Attack Bonus)/(100 + Magic Defense Bonus)
- Malgré l'avantage du Chevalier, l'impact est moins significatif que la Magic Evasion élevée du Sakpata

### 3. INT (Différence +21 pour Chevalier)

- Affecte la Magic Accuracy ennemie selon le dINT (différence INT)
- Impact sur Magic Accuracy :
  - dINT < 10 : 1 INT = 1 Magic Accuracy
  - dINT > 10 : 2 INT = 1 Magic Accuracy
  - dINT > 30 : 4 INT = 1 Magic Accuracy

## Conclusion

La supériorité du set Sakpata pour la résistance magique s'explique principalement par sa Magic Evasion significativement plus élevée (+60). Cette statistique, combinée avec l'Elemental Resist (15) et le Carrier's Sash, permet d'obtenir plus fréquemment des résistances totales (1/8 des dégâts), surpassant les avantages du set Chevalier en Magic Defense Bonus (+11) et INT (+21).

La différence de dégâts moyens reçus (49.84 vs 98.04) illustre clairement l'efficacité supérieure de la Magic Evasion par rapport aux autres stats défensives magiques dans ce contexte spécifique.

Pour une résistance magique optimale, privilégier une Magic Evasion élevée combinée à de l'Elemental Resist s'avère plus efficace qu'un focus sur Magic Defense Bonus et INT.
