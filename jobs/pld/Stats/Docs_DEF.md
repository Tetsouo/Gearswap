### Résumé Global sur la DEF pour Paladin

La DEF (Défense) est un élément essentiel pour réduire les dégâts physiques subis, jouant un rôle clé dans la survie du Paladin. Elle complète les caps globaux comme PDT (Physical Damage Taken) et DT (Damage Taken), offrant une réduction supplémentaire des dégâts physiques, en particulier dans des contenus où les ennemis possèdent une Attaque élevée.

---

### Fonctionnement de la DEF

#### Composants de la DEF

1. **DEF de base** :

   - Dépend du niveau du joueur et de sa Vitalité (VIT).
   - **Formule** : `DEF de base = floor(VIT × 1.5) + ajustement selon le niveau`

2. **DEF des équipements** :

   - Ajout direct des valeurs de DEF fournies par l'équipement.

3. **Buffs actifs** :
   - Augmentations temporaires comme **Protect V** ou **Cocoon**.

#### Impact de la DEF

La DEF est comparée à l’Attaque de l’ennemi pour déterminer le ratio Attaque/DEF :

- **Ratio favorable (DEF > Attaque)** : Les dégâts subis sont réduits.
- **Ratio défavorable (DEF < Attaque)** : Les dégâts augmentent proportionnellement à l'écart entre l'Attaque de l'ennemi et la DEF du joueur. Cependant, une DEF élevée peut limiter l'ampleur de cet écart.

##### Exemple chiffré :

Si l'Attaque d'un ennemi est de 1200 et que la DEF du Paladin est de 1000 :

- Le ratio Attaque/DEF est 1.2, entraînant des dégâts amplifiés.
- Si la DEF est augmentée à 1300, le ratio devient 0.92, réduisant ainsi les dégâts subis.

---

### Buffs et Optimisation de la DEF

#### Buffs indispensables

1. **Protect V** :

   - **Effet** : Ajoute **220 DEF** au joueur.
   - Lorsqu'il est lancé par un Paladin à partir du niveau 70 avec le trait **Shield Barrier**, la DEF du bouclier équipé est ajoutée à cet effet.
   - **Exemple** : Avec un bouclier ayant 130 DEF, **Protect V** fournit 350 DEF (220 + 130).

2. **Cocoon** (Blue Mage) :
   - Augmente la DEF totale de **50 %** pendant une courte durée.

#### Équipements recommandés

- **Priorité** : Favorisez les équipements offrant à la fois une DEF élevée et une augmentation de la VIT (puisque 1 point de VIT = 1.5 DEF).
- Exemples :
  - Armures avec des stats DEF importantes.
  - Bijoux et accessoires augmentant la VIT.

#### Priorité tactique

- Une fois les caps PDT/DT atteints, la DEF devient un outil précieux pour réduire encore davantage les dégâts physiques.
- Complétez la DEF avec des buffs d'équipe (comme Protect) et des sorts.

---

### Utilisation Pratique

#### Quand privilégier la DEF ?

1. **Contre des ennemis à Attaque élevée** :

   - Contenus comme Odyssey ou Dynamis Divergence, où maintenir un ratio Attaque/DEF raisonnable est crucial.

2. **Après avoir atteint les caps PDT/DT** :
   - La DEF joue un rôle complémentaire mais significatif.

#### Points importants

- La DEF seule ne suffit pas pour une mitigation optimale.
- Elle doit être utilisée en synergie avec les caps PDT et DT pour une réduction des dégâts efficace.

---

### Conclusion

La DEF est une statistique importante pour tout Paladin, en particulier dans des contenus où les ennemis infligent des dégâts physiques élevés. Une fois les caps PDT et DT atteints, investir dans une DEF élevée offre une réduction supplémentaire des dégâts, renforçant le rôle du Paladin en tant que tank principal.
