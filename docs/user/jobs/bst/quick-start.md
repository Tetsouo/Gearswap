# BST - Quick Start Guide

## Loading the System

1. **Change to BST in-game**
2. **Load GearSwap**:

   ```
   //lua load gearswap
   ```

3. **Verify loading**:
 - Look for: `[BST] Functions loaded successfully`
 - Keybinds auto-loaded
 - Macrobook set to Book 1, Page 1
 - Lockstyle #6 applied after 8 seconds

---

## Verify System Loaded

**Check keybinds loaded**:

```
>> Console shows: "BST SYSTEM LOADED"
```

**Check UI (if enabled)**:

```
//gs c ui
>> Shows all keybinds overlay
```

**Validate equipment**:

```
//gs c checksets
>> Shows item validation (VALID/STORAGE/MISSING)
```

---

## First Commands to Try

```
//gs c checksets          # Validate your equipment
//gs c ui                 # Toggle keybind display
//gs reload               # Reload system
```

---

## Default Setup

On load, system automatically:

- Loads all BST keybinds (11 total)
- Sets macrobook (Book 1, Page 1 for SAM)
- Applies lockstyle #6 (after 8s delay)
- Displays UI with keybinds (if enabled)
- Sets default states:
 - Default states loaded
