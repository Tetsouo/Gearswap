# PUP -- Puppetmaster

Automaton management with pet precast and midcast handling.

## Modules

The PUP implementation includes 14 modules (12 standard + PUP_PET_PRECAST and PUP_PET_MIDCAST for automaton-specific gear logic).

## Setup

1. Copy `_master/entry/Tetsouo_PUP.lua` to `[YourName]/[YourName]_PUP.lua`
2. Copy `_master/config/pup/` to `[YourName]/config/pup/`
3. Copy `_master/sets/pup_sets.lua` to `[YourName]/sets/` and fill in your gear

## Commands

Uses the standard command set. See [commands reference](../../guides/commands.md).
