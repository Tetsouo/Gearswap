# RUN -- Rune Fencer

Tank job with rune management, ward system, and BLU magic support for enmity.

## Modules

Standard 12-module setup plus rune management and cure set builder logic modules.

## Setup

1. Copy `_master/entry/Tetsouo_RUN.lua` to `[YourName]/[YourName]_RUN.lua`
2. Copy `_master/config/run/` to `[YourName]/config/run/`
3. Copy `_master/sets/run_sets.lua` to `[YourName]/sets/` and fill in your gear

## Commands

Uses the standard command set. See [commands reference](../../guides/commands.md).
